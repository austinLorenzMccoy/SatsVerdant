# SatsVerdant

Turn Trash into Bitcoin - A sustainable platform that rewards users with sBTC for verified recycling activities.

## 🌱 Overview

SatsVerdant is a proof-of-concept platform that demonstrates how blockchain technology can drive environmental positive impact. Users submit their recycling activities, get them verified, and earn synthetic Bitcoin (sBTC) rewards.

## 🏗️ Project Structure

```
mvp_folder/
├── backend/          # FastAPI Python backend
├── frontend/         # Next.js TypeScript frontend  
├── contracts/       # Smart contracts
├── ml-training/      # Machine learning models
└── docs/           # Documentation
```

## 🚀 Tech Stack

- **Frontend**: Next.js 16, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: FastAPI, Python
- **Blockchain**: Bitcoin, Stacks
- **ML**: TensorFlow, PyTorch

## 📦 Quick Start

### Frontend
```bash
cd frontend
pnpm install
pnpm dev
```

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
